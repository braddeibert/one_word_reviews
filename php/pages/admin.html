  • Function Number: 1

    Function Name: Search Content by Title*

    URL: http://webdev.cs.umt.edu/~bd152220/one_word_reviews/php/pages/result.php

    Function Type: Query

    Description: Searches the CONTENT table in the database for all titles matching a search term. Results are pulled and listed under the 'Content' header.

    Primary Users: Site users (reviewers/readers)

    Tables affected: CONTENT

    Sample Input: Search term (e.g. 'The Dark Knight')

    Sample Output: List of matching titles, along with their producer and year released. Sorted by title.

    Notes for use: Type your search into the search form atop the page in the navbar, and submit.

    Error handling: Searches with no matches meet a 'No results found' message. An empty search displays all contents in the CONTENT, REVIEWS, and USERS tables.

  • Function Number: 4

    Function Name: Post Review*

    URL: http://webdev.cs.umt.edu/~bd152220/one_word_reviews/php/pages/review.php

    Function Type: Query and Insert

    Description: Posts new reviews to existing content in the database. Submissions must be authored by an existing user in the database.

    Primary Users: Site users (reviewers/readers)

    Tables affected: REVIEWS, CONTENT, USERS

    Sample Input: Content title, one word review, and author's username. (e.g. 'Shrek', 'AWESOME', 'BIGTIMETOMMIE1')

    Sample Output: Re-routes to a page confirming a successful submission.

    Notes for use: Hit 'Write a Review' at the bottom of the page. Fill out the form that appears, and submit.

    Error handling: Form will not submit until all fields are filled. Users will receive error messages if any of the fields is incorrectly filled out.

  • Function Number: 5

    Function Type: Insert

    Description: Creates a new user and adds them to the USERS table.

    Primary Users: Site users (reviewers/readers)
